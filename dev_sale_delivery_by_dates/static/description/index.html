Sale Delivery by Date
For Helping to create different delivery order for sale order based on given delivery date on sale order line

DevIntelle Consulting Service Pvt.Ltd.

Features
  • In Business, Many times we having one sale order with number of products but the delivery of those products on different dates. so to come soluation Devintelle have create a custom module/Application which create different delivery order for sale order based on given delivery date on sale order line
  • Different delivery order for sale order
  • Easy to use with no configuration

Setup & Configure

Select Delivery Date

In sale order line Screen Select Delivery Date when the product will be deliver.

Confirm the Sale Order

By Confirming Sale order, Delivery Order created based on the sale order line delivery dates.

View Delivery Order

In above case 2 delivery order is created one delivery order for delivery date 07/14/2017 and second delivery order for delivery date 07/14/2017 products
